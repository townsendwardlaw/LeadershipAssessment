The Build Arc

Four phases, twelve sessions. AI wasn't just writing code — it was thinking through architecture, UX, pricing, copy, training design, and go-to-market strategy alongside a human builder.

Phase 1 Architecture & Knowledge System Sessions 1–3
Diagnosis
Identified why the original GPT was failing — straying from methodology, improvising generic sales advice, making things up
Architecture
Designed three-layer system: System Prompt (routing) + Governing Protocols (rules) + Knowledge Base (methodology)
Content
Scoped 13 methodology components from Outreach through Renewals — defined four operating modes (Learn, Train, Debrief, Plan)
Gap Analysis
Found and filled 6 knowledge gaps — Discovery language, multiple sessions guidance, attachment principle, written proposals, and more
Simulation Design
Built role-play setup system — 5 context questions with validation logic so AI simulations are grounded in real prospect data
Decision
Retired broken frameworks — killed the incomplete "Five Outcomes of a Connect Call" instead of patching it
Phase 2 Product & Platform Sessions 4–5
Technical
Web app architecture decisions: Claude API backend, Supabase auth, conversation-only persistence, Stripe billing
Pricing
Set pricing at $97/mo and $997/yr — anchored to ROI: "one new client pays for two full years"
Guarantee
90-day better-than-money-back + $100 — verifiable through AI conversation logs. Real teeth in both directions.
Messaging
Core value prop: "Most coaches don't have a marketing problem. They have an enrollment problem."
Phase 3 UX, Design & Training Sessions 6–11
Landing Page
Built 3 versions of the landing page — each iteration informed by UX critique, real testimonials, and scenario-based messaging
UX System
Built a "Virtual Ward" knowledge base — 50+ articles, 17 case studies, design frameworks — then applied it to critique the product
Training
60-min Discovery training session designed — teaching structure, common mistakes woven into pillars, self-diagnostic questions
Onboarding
Complete onboarding spec: 5-step flow, skip logic, context capture, methodology-rooted routing to first action
UI/UX
Four friction points identified and solved: mode switching, methodology trust, mobile usability, onboarding depth
Testimonials
15 testimonials polished from 13 beta users — organized into 3 tiers with before/after framing
Phase 4 Launch Preparation Session 12+
Stripe
Payment infrastructure built: products, prices, trial config, coupon codes, payment links — all in test mode, ready to flip live
Email
11-email migration series drafted — transitioning GPT users, Kit broadcasts scheduled, Gmail drafts created
Deployment
Landing page deployed to Netlify — domain strategy set: how2sellcoaching.com (sales) → clientcreationcopilot.com (app)
System
System Prompt iterated to v2.4 — removed KB component refs, added plain prose formatting, GPT transition notice posted

This wasn't AI writing code while a human watched. It was a methodology that lived in one person's head being systematically extracted, structured, validated, designed around, and prepared for market — with AI as a thinking partner at every stage. Architecture. UX. Pricing. Copy. Training design. Go-to-market. All of it.
